# 2DRakun

This application was developed on request by a friend who needed a simple and fast way to create invoices for his company. The goal is to help small businesses generate invoices quicker and easier than using Excel templates, with added support for a 2D barcode to facilitate payments.

## Features

- User registration with mandatory payment details (IBAN, etc.)
- Adding and automatically saving customer data when creating invoices
- Selecting previously saved customers to auto-fill data or entering new customer info for each invoice
- Automatic customer data updates on every invoice submission if changes occur
- Invoices saved with a 2D barcode for easy payment processing
- Access to previously created invoices

## Technologies

- **Backend:** ASP.NET MVC  
- **Frontend:** Vue.js (initial implementation)  
- **Database:** SQL Server

## Development Phases

### Phase 1

- Implementation of core functionality using Vue.js frontend and ASP.NET backend  
- Focus on fast and easy invoice and customer management

### Phase 2

- Migration of the frontend to React as part of learning React and enhancing user experience
